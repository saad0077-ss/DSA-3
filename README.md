# 📘 DSA-3 Repository

This repository contains my practice and implementations of **Data Structures and Algorithms (DSA)** in Dart.  
It is part of my portfolio to showcase problem-solving skills and coding proficiency.

---

## 📂 Topics Covered

### 🌳 Trees
- Introduction to Trees
- Types of Trees (Binary Tree, N-ary Tree, etc.)
- Tree Traversals (Preorder, Inorder, Postorder)

### 🌲 Binary Search Tree (BST)
- Insertion, Deletion, Search
- Validating BST
- Finding Closest Value
- Traversals

### 🔡 Trie
- Insert and Search operations
- Prefix Matching
- Autocomplete functionality
- Word Count and Deletion

### ⛰️ Heap
- Min Heap and Max Heap
- Heap Operations (Build, Insert, Remove)
- Applications in Priority Queues

### ⚡ Heap Sort
- Sorting using Heap
- Time Complexity: O(n log n)
- In-place Sorting

### 🔗 Graphs
- Representation (Adjacency List / Matrix)
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Applications: Shortest Path, Cycle Detection, Connected Components
